# BrightTV_-Viewership-Analytics-
BrightTV is a media analytics project designed to explore user demographics, viewership patterns, and channel engagement.
The dataset includes user profiles and viewer transactions, enabling comprehensive analysis of audience behavior and content performance.
Dataset Components
1. User Profiles
•	UserID: Unique identifier for each user
•	Gender: Male, Female, Unknown
•	Race: Demographic classification
•	Age: Numeric value, grouped into categories (Kids/Teens, Youth, Adults, Seniors)
•	Province: Geographic location of the user
2. Viewer Transactions
•	UserID0: Links to User Profiles
•	Channel: TV channel watched
•	RecordDate: Date and time of viewing
•	Duration: Length of viewing session (in minutes or seconds)

Key Insights
•	Evening prime-time drives the highest engagement.
•	Adults (35–55) form the largest audience segment.
•	Certain provinces dominate viewership, offering regional targeting opportunities.
•	Kids/Teens and Seniors represent smaller but distinct segments needing tailored content.
Usage
•	Run SQL queries to clean and transform data.
•	Use Databricks/Spark SQL for integration and analysis.
•	Export results into dashboards for visualization.
Dependencies

•	Visualization tools (e.g., Power BI, Tableau)
